# Buddy Apps Website

A static GitHub Pages website for Buddy Apps.

## Site structure
- `/` — company homepage describing the business and product-development approach
- `/products/hangover-buddy/` — Hangover Buddy product page and store links
- `/products/thermovert/` — Thermovert product page and recipe-conversion workflow
- `/support/` — product support form
- `/privacy/` — privacy policy

## Purpose
Buddy Apps designs focused mobile apps around specific, underserved everyday needs.

## Deploy on GitHub Pages
1. Create a GitHub repository.
2. Upload these files.
3. Go to Settings → Pages.
4. Set the source to the `main` branch and root folder.
5. Save.

## Search indexing
Submit `https://ideabuddies.com/sitemap.xml` in Google Search Console after deployment.
