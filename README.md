# Buddy Apps Website

A simple static GitHub Pages website for Buddy Apps.

## Included Pages
- `index.html`
- `styles.css`
- `images/hero.jpg`
- `images/hangover-buddy.png`

## Purpose
Buddy Apps creates simple app companions that help people make better decisions and achieve their goals.

## First App
Hangover Buddy reminds someone who is drinking alcohol to follow practical best practices such as hydration, pacing, and food check-ins.

## Deploy on GitHub Pages
1. Create a GitHub repository.
2. Upload these files.
3. Go to Settings → Pages.
4. Set source to the `main` branch and root folder.
5. Save.

## Replacing Images
Replace these files while keeping the same names:
- `images/hero.jpg`
- `images/hangover-buddy.png`
